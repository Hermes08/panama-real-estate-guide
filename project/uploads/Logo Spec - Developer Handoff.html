01 · Overview

Two variants. Both share the same sticker-style .com chip — the chip is the fixed brand element and never changes. What changes is how Panama, RealEstate, and Guide are treated.

When to use which

PanamaRealEstateGuide.com

VARIANT H · PRIMARYInk Block

Default master logo. Use everywhere unless you specifically need the quieter alternate. Favicon, navbar, emails, business cards, YouTube thumbnails, OG share images, print.

PrimaryNavbar · marketing · YouTube thumbs Min width180px
PanamaRealEstateGuide.com

VARIANT I · ALTERNATECoral Underline

Editorial alternate. Use in text-heavy contexts — email signatures, press, blog-post bylines, long-form guides — where the Variant H ink blocks would feel too graphic.

SecondaryEditorial · email signature · blog bylines Min width160px

05 · Usage rules — do & don't

These are hard rules. The .com chip is non-negotiable brand equity; protect it.

✓ Do
  • Use Variant H (Ink Block) as the default everywhere.
  • Use Variant I (Coral Underline) only in editorial/text-heavy contexts.
  • Keep clearspace ≥ the height of the ink block on all sides.
  • Use the -dark modifier on backgrounds darker than 50% L*.
  • Preserve the .com chip's -2° rotation and hard ink/cream shadow.
  • Always wrap the whole lockup in an <a href="/"> linking home.
✗ Don't
  • Don't change the chip color — coral is the fixed brand anchor.
  • Don't straighten the chip or remove its drop shadow.
  • Don't mix variants (H bookends + I underline) in one lockup.
  • Don't use on gradients or busy photos without a solid color plate behind.
  • Don't rewrite the domain — never "PanamaRealEstate.com" or "PREG.com".
  • Don't substitute fonts. Manrope, always.
